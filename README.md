A lightweight TypeScript SDK for NASA's [Astronomy Picture of the Day (APOD)](https://api.nasa.gov), featuring:

- `getApod`, `getRandom`, `getRange` methods  
- Automatic retry & rate-limit handling  
- React-friendly `AbortSignal` support  
- Full TypeScript types  
- Fully tested with Vitest
